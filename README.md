# React ML4U

## Installing

- Clone this repository:

```
git clone https://github.com/toannhu/ML4U_Frontend/
```

- How to run locally?

1. cd into the repo.
2. Run `yarn` or `npm i` to download dependencies.
3. Run `yarn start` or `npm start` to start the development server.
